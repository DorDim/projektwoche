Schritt-für-Schritt

  1. Auf dem Zielrechner Python 3 installieren.
  2. Repository klonen (oder ZIP herunterladen und entpacken), dann in den Projektordner wechseln.
  3. Windows (empfohlen): PowerShell als Benutzer öffnen.
  4. Virtuelle Umgebung anlegen und Abhängigkeiten installieren (siehe Befehle unten).
  5. Den Hintergrund-Task mit Skript erstellen; der Agent startet danach automatisch ohne offenes Terminal.
  6. Im Dashboard prüfen, ob der neue Client erscheint.
