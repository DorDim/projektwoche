Anwenderdokumentation

Diese Seite ist als praxisnahe Schritt-für-Schritt-Anleitung aufgebaut. Sie unterstützt beim Einstieg, beim täglichen Betrieb und bei typischen Fehlerfällen.

Schnellstart Szenarien Rollen & Rechte Troubleshooting

Schnellstart in 5 Minuten

  1. Mit Benutzername und Passwort anmelden.
  2. Im Dashboard einen Client auswählen.
  3. Zeitspanne auf 24h setzen.
  4. Durchschnittswerte, Alerts und Laufwerksstatus prüfen.
  5. Bei Bedarf über Vergleich und Export weiter analysieren.
Tipp: Wenn Daten fehlen, zuerst auf Aktualisieren klicken.

Rollen und Berechtigungen

Sichtbare Menüpunkte und Aktionen hängen von den Benutzerrechten ab.

Recht Bedeutung
Dashboard ansehen Zugriff auf Dashboard und Vergleich.
Clients hinzufügen/löschen Geräte Onboarding und Entfernen.
Nutzer verwalten Benutzerkonten erstellen, ändern und deaktivieren.
Events ansehen Ereignisprotokoll und Systemhinweise einsehen.
Alert-Regeln verwalten Grenzwerte für Warnungen konfigurieren.

Szenario A: Admin richtet Team ein

Benutzer anlegen und Rechte sauber verteilen.

  1. Als Admin anmelden.
  2. Nutzer-Seite öffnen.
  3. Benutzer mit Rolle und Rechten anlegen.
  4. Test-Login durchführen.

Szenario B: Client Onboarding

Voraussetzung: Recht Clients hinzufügen.

  1. „Neuen Client hinzufügen“ öffnen.
  2. Token kopieren.
  3. Ein-Befehl auf Zielsystem ausführen.
  4. Dashboard aktualisieren.

Szenario C: Tägliche Überwachung

  1. Client auswählen.
  2. Zeitspanne festlegen.
  3. Durchschnittswerte und Diagramme prüfen.
  4. Auffälligkeiten und Alerts bewerten.
  5. Laufwerke (Gesamt/Frei) prüfen.

Szenario D: Vergleich und Export

Vergleich

  1. Vergleich-Seite öffnen.
  2. Clients auswählen.
  3. Kennzahlen und Diagramm auswerten.

Export

  1. Client auswählen.
  2. Format (JSON/CSV/PDF) wählen.
  3. Datei speichern.

Wichtige Hinweise

Demo-Daten sind nur mit dem Demo-Benutzer sichtbar.
Fehlende Menüpunkte bedeuten in der Regel fehlende Berechtigungen.

Troubleshooting

Login fehlgeschlagen

  • Benutzername/Passwort prüfen
  • Groß-/Kleinschreibung beachten
  • Benutzerkonto aktiv?

Keine Clients sichtbar

  • Auf „Aktualisieren“ klicken
  • Agent auf Zielsystem prüfen
  • Recht „Dashboard ansehen“ vorhanden?

Nutzer-Seite fehlt

  • Recht „Nutzer verwalten“ fehlt
